🧠 QuizMania – Interactive Quiz Web App
🚀 Overview

QuizMania is a modern, responsive quiz web application designed to test and improve users’ knowledge across multiple categories. It provides an engaging experience with instant feedback, scoring, and a clean user interface.

✨ Features
📚 Multiple quiz categories (Science, Mathematics, General Knowledge)
✅ Instant feedback for each answer
📊 Score tracking with performance analysis
👤 User name input before starting quiz
📱 Fully responsive (mobile-first design)
💾 Local storage for saving score
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
QuizMania/
│── index.html
│── style.css
│── script.js
▶️ How to Run
Clone or download this repository
Open index.html in any web browser
Enter your name and start the quiz
🎯 Purpose

This project showcases frontend development skills including UI design, DOM manipulation, event handling, and building an interactive web application.
